colosseum

Tao Colosseum is a decentralized peer-to-peer betting protocol built on the Bittensor EVM, featuring an 'underdog' game where participants bet on either the red or blue team, and the side with less total stake wins and shares the pool. The platform incentivizes participation by distributing rewards to miners based on their time-decayed betting volume over a 7-day window, with recent activity weighted more heavily. Validators query the smart contract to track betting volumes and set network weights, while features like verifiable on-chain randomness, anti-sniping mechanisms, and cryptographic wallet mapping enhance fairness and security. The protocol includes a leaderboard, REST API for validator data, and supports seamless integration with the Bittensor metagraph.
