Compute Horde

computehorde is a specialized subnet (subnet 12) within the Bittensor network, designed to provide scalable, decentralized, and trusted GPU computing power for validators across other Bittensor subnets. By transforming untrusted GPUs from miners into reliable compute resources, computehorde reduces dependency on centralized services and enables cost-effective, large-scale validation. Key features include fair and verified work mechanisms, scalable mining with multiple executors per miner, hardware class-based resource allocation, a developer SDK for easy integration, collateral-based trust for organic jobs, and an optional DDoS shield for miners. The project aims to support a wide range of GPU types and offers a robust scoring and incentive system to ensure quality and accountability.
